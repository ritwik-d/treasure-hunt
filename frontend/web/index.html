Treasure Hunt

Incredible Gameplay

Treasure Hunt is a mobile application designed to get players excercise while simultaneously having fun. Players create and find virtual treasure which is conveniently placed on an interactive map. This concept is extended to incorporate several features including races, groups, and much more!

Features

Our features deliver an immersive experience to Treasure Hunt.

Create & Find Treasure

The objective of Treasure Hunt is for players to create and find virtual treasure, know as "challenges." When a player places virtual treasure, others can see it on a map and physically travel to it (by walking, biking, or running).

Private Groups

Players have the ability to create groups with other players. In groups, players can customize their challenges to only be visible to certain groups. Other conveniences are included to invoke friendly competition including a leaderboard, a chat, etc.

Races (BETA incomplete)

Along with challenges, players can find treasure through races. In races, players are in a "race" to find the treasure before the other racers. Players can see where other racers are, and which racers are closest to finding the treasure.
